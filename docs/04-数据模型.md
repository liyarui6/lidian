# 04 - 数据模型

## 数据库配置

| 配置项 | 值 |
|--------|-----|
| 数据库名称 | `XiaoDianDB` |
| 版本 | `1` |
| 对象仓库名称 | `transactions` |
| 主键 | `id`（自增） |

## Transaction（交易记录）

| 字段 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| `id` | number | 是 | 自增 | 唯一标识 |
| `type` | string | 是 | - | `'income'` 或 `'expense'` |
| `amount` | number | 是 | - | 金额，正数，最多两位小数 |
| `category` | string | 是 | - | 用途分类 |
| `paymentMethod` | string | 否 | `''` | 支付方式（仅支出时填写，收入时为空） |
| `note` | string | 否 | `''` | 备注 |
| `date` | string | 是 | 当天 | 日期，格式 `YYYY-MM-DD` |
| `createdAt` | number | 是 | 当前时间戳 | 创建时间（毫秒） |

## 索引

| 索引名 | 字段 | 说明 |
|--------|------|------|
| `dateIdx` | `date` | 用于按日期快速查询和排序 |

## 枚举值

### type（交易类型）
| 值 | 含义 |
|----|------|
| `'income'` | 收入 |
| `'expense'` | 支出 |

### category（用途分类）

**支出类：**
`'餐饮'`, `'交通'`, `'购物'`, `'娱乐'`, `'住房'`, `'医疗'`, `'其他'`

**收入类：**
`'工资'`, `'兼职'`, `'理财'`, `'红包'`, `'报销'`, `'其他'`

### paymentMethod（支付方式）
`'微信'`, `'支付宝'`, `'现金'`, `'银行卡'`, `'其他'`

## 示例数据

```json
{
  "id": 1,
  "type": "expense",
  "amount": 35.50,
  "category": "餐饮",
  "paymentMethod": "微信",
  "note": "午餐外卖",
  "date": "2026-06-22",
  "createdAt": 1719032400000
}
```
