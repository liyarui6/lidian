# 06 - 部署指南

## 部署到 GitHub Pages

### 前提条件
- 一个 GitHub 账号（免费注册：github.com）
- 本项目代码已准备好

### 步骤

#### 1. 创建 GitHub 仓库
1. 打开 https://github.com 并登录
2. 点击右上角「+」→「New repository」
3. 仓库名填写：`xiaodian-jizhang`（或你喜欢的名字）
4. 选择 Public（公开）
5. 不要勾选任何初始化选项
6. 点击「Create repository」

#### 2. 上传代码
在项目根目录依次执行以下命令（由AI协助执行）：

```bash
git init
git add .
git commit -m "初始化小点记账项目"
git branch -M main
git remote add origin https://github.com/你的用户名/xiaodian-jizhang.git
git push -u origin main
```

#### 3. 开启 GitHub Pages
1. 打开仓库页面，点击「Settings」
2. 左侧菜单点击「Pages」
3. 「Source」选择「Deploy from a branch」
4. 「Branch」选择「main」，文件夹选择「/ (root)」
5. 点击「Save」
6. 等待1-2分钟，页面会显示部署后的网址
7. 网址格式：`https://你的用户名.github.io/xiaodian-jizhang/`

#### 4. 手机端安装到桌面
1. 安卓手机打开 Chrome 浏览器
2. 输入上述网址并打开
3. 等待页面加载完成
4. 点击浏览器右上角菜单（三个点）
5. 选择「添加到主屏幕」
6. 确认添加
7. 桌面出现「小点记账」图标

### 常见问题

**Q: 添加到主屏幕的选项没出现？**
A: 多打开几次，等待浏览器缓存完成。或者清除浏览器缓存后重试。

**Q: 图标显示不对？**
A: PWA图标会在第二次打开时正确显示，第一次可能显示为网页截图。

**Q: 想更新App怎么办？**
A: 推送新代码到GitHub后，GitHub Pages会自动更新。手机App会在联网时自动更新（通常24小时内）。

**Q: 更换手机了数据怎么办？**
A: 目前数据存储在手机本地，更换手机数据不会自动迁移。后续版本会考虑导出功能。
