# 03 - 设计规范

## 色彩系统

| 用途 | 色值 | 说明 |
|------|------|------|
| **主色** | `#F9A825` | 金黄色，用于导航栏、按钮、强调元素 |
| **主色深** | `#F57F17` | 深金色，用于按钮按下态 |
| **主色浅** | `#FFD54F` | 浅金色，用于高亮背景 |
| **背景色** | `#FFF8E1` | 极浅黄色，用于页面背景 |
| **卡片背景** | `#FFFFFF` | 白色，用于卡片和表格行 |
| **收入色** | `#4CAF50` | 绿色，收入金额和标签 |
| **支出色** | `#F44336` | 红色，支出金额和标签 |
| **文字主色** | `#212121` | 深灰黑，正文 |
| **文字辅色** | `#757575` | 灰色，次要信息 |
| **分隔线** | `#E0E0E0` | 浅灰，分隔线和边框 |

## 字体

- 系统默认中文字体（不引入Web字体，减小体积）
- 金额数字使用粗体
- 字号层级：
  - 页面标题：20px
  - 汇总金额：28px（粗体）
  - 表格表头：14px（灰色）
  - 表格内容：15px
  - 辅助文字：12px

## 圆角

- 大卡片：12px
- 小卡片/按钮：8px
- 弹窗顶部：16px（仅顶部两角）
- 表格行：无圆角（扁平）

## 间距

- 页面边距：16px
- 卡片内边距：16px
- 组件间距：12px
- 表格行高：48px
- 图标与文字间距：8px

## 组件规范

### 底部导航栏
- 高度：56px
- 三个等宽Tab：今日 / 统计 / 记录
- 选中态：主色文字 + 主色指示点
- 未选中：灰色文字
- 背景：白色，顶部1px分隔线

### 记账按钮（FAB）
- 圆形，直径56px
- 背景色：主色 `#F9A825`
- 图标：白色「+」，字号28px
- 位置：右下角，距底部72px（给导航栏留空），距右边16px
- 阴影：`0 4px 12px rgba(249, 168, 37, 0.4)`

### 记账弹窗
- 从底部滑入，覆盖80%屏幕高度
- 顶部有拖动条（灰色小横条）
- 背景白色，顶部圆角16px
- 遮罩层半透明黑色

### 表格
- 表头：浅黄背景 `#FFF8E1`，灰色文字
- 数据行：白色背景，底部1px分隔线
- 列宽分配（移动端320px基准）：
  - 用途列：25%
  - 金额列：25%
  - 支付列：22%
  - 备注列：28%

### 汇总卡片
- 白色卡片，圆角12px，阴影轻微
- 水平分为两栏：收入（左，绿色）| 支出（右，红色）
- 下方一行：净收入（居中，正数绿色/负数红色）

### 条形图
- 白色卡片容器，圆角12px
- 每个时间单位一组，收入（绿色）和支出（红色）并排
- 柱体宽度14px，圆角顶部
- 高度按比例缩放，最高柱100px
- 底部图例：绿点=收入，红点=支出
- 日维度：近7天；周维度：近4周；月维度：近6个月

## 移动端适配

- 最大宽度：480px（超过后居中显示，两侧留灰底）
- 最小宽度：320px
- 视口设置：`viewport width=device-width, initial-scale=1.0`
- 所有可点击区域最小 44x44px（拇指友好）
- 表单输入框高度 48px
