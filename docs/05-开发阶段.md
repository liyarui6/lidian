## 阶段2：数据层 + 记账弹窗

### 任务
- 实现IndexedDB初始化（打开数据库、创建对象仓库和索引）
- 实现增删查改函数（addTransaction、deleteTransaction、getTransactions）
- 实现记账弹窗HTML和CSS
- 实现弹窗交互（打开/关闭/类型切换/分类切换）
- 实现表单验证（金额必填且为正数）
- 实现保存逻辑（写入IndexedDB）

### 产出物
- `src/index.html`（JS数据层 + 弹窗部分）

### 验收标准
- [ ] 数据库能正常打开
- [ ] 弹窗点击FAB能打开
- [ ] 支出/收入切换后分类自动变化
- [ ] 保存成功，浏览器DevTools Application > IndexedDB中能看到数据
- [ ] 金额为空时提示错误
