# 01 - 需求文档

## 项目信息

| 项目 | 内容 |
|------|------|
| **项目名称** | 小点记账 |
| **目标平台** | 安卓手机（通过PWA实现） |
| **网络要求** | 完全离线，无需联网 |
| **目标用户** | 普通用户，非技术背景 |

---

## 功能需求

### P0 - 核心功能（必须实现）

| 编号 | 功能 | 说明 |
|------|------|------|
| F1 | **记账** | 记录一笔收支，包含：类型、金额、用途分类、支付方式、备注、日期 |
| F2 | **今日概览** | 主界面显示今日收入总额、支出总额、结余 |
| F3 | **今日记录表格** | 主界面以表格形式展示今日所有记录 |
| F4 | **本地存储** | 所有数据存于手机本地（IndexedDB），无需服务器 |
| F5 | **固定分类** | 提供预设的收支分类供用户选择 |

### P1 - 重要功能

| 编号 | 功能 | 说明 |
|------|------|------|
| F6 | **日/周/月统计** | 按日、周、月查看收支汇总 |
| F7 | **分类统计** | 查看每个分类的消费金额和占比 |
| F8 | **历史记录** | 按月查看所有历史记录 |
| F9 | **删除记录** | 支持删除某条记录 |

### P2 - 增强功能（可选，后期迭代）

| 编号 | 功能 | 说明 |
|------|------|------|
| F10 | 数据导出 | 导出为CSV或图片 |
| F11 | 搜索记录 | 按备注或金额搜索 |
| F12 | 编辑记录 | 修改已保存的记录 |

## 非功能需求

| 需求 | 说明 |
|------|------|
| **离线可用** | 无网络时所有功能正常使用 |
| **响应速度** | 页面操作响应 < 200ms |
| **数据安全** | 数据仅存本地，不上传任何服务器 |
| **安装体验** | 浏览器打开即可「添加到主屏幕」，像原生App |
| **界面适配** | 适配主流安卓手机屏幕（360px-428px宽度） |
